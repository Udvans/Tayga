# Tayga
Java- лаунчер для игрулек.
В теории, должен будет законнектиться с бд на сервере, и скачивать оттуда игры.
---
# Прогресс: 
* Вход в аккаунт  -
* Создание аккаунта -
* Создание профиля -
* Поиск игр по названию -
* Филирация игр по жанрам -
* Сохранение игр на пк -
* Добавление игр в бд -
* Редактирование своих игр в бд -
